## Project Description

The Electrical Appliance Load Monitoring System is a C++ program that helps users monitor electricity usage of household appliances.

The system allows users to register appliances and view the list of registered appliances.

## Week 1 Implementation

In Week 1, the following features were implemented:

- Project structure creation
- Appliance data structure using `struct`
- Menu driven system
- Register appliance feature
- View appliances feature
- Use of `vector` to store appliance data
