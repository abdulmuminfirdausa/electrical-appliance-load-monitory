## Project Overview
This is a C++ console application that registers electrical appliances, calculates daily energy usage (kWh/day), and estimates electricity cost using a user-provided tariff. The project also supports file storage so appliance data is saved and can be loaded again when the program restarts.

## Features
- Register appliances (name, power in watts, hours used per day)
- View all registered appliances
- Search appliances by name (case-insensitive)
- Display daily energy summary (kWh/day)
- Generate a billing summary report (daily + monthly estimates)
- Save/load appliances using text files
- Export billing report to `billing_summary.txt`

## Energy & Billing Formula
**Daily Energy (kWh/day):**
`kWh/day = (Power(W) × Hours/day) / 1000`

**Daily Cost:**
`Cost/day = Total(kWh/day) × Tariff`

**Monthly Estimate (30 days):**
`Monthly Cost = Daily Cost × 30`
